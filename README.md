# mqtt_installer

## Paho mqtt Lib Installation

```bash
./install_paho_mqtt_c.sh
./install_paho_mqtt_cpp.sh
```

## Code demonstration

```bash
cd app
mkdir build
cd build
cmake ..
make
```

## Execution

**terminal #1:**

```bash
./publish
```

**terminal #2:**

```bash
./consume
```
